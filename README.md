未来还将细致实现点赞、反对、编辑、删除等操作。

未来一定会支持多行文字的输入。

未来一定会支持用上下左右键实现选择选项/要回复的评论。

未来会使用cmake，但我现在还不很会这个。

未来会将部分写好的很少会再动的文件做成lib，减少编译时间（现在编译一次要差不多十五秒钟真是太慢了吧）。

未来可能会搭建服务器以及抛弃控制台界面。
